## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 开发环境
```bash
npm run dev
```

### 生产构建
```bash
npm run build
```

### 部署
```bash
docker build -t eaglercraft:v2.0 .
docker run -p 8080:8080 eaglercraft:v2.0
```

## 📝 开发文档

详细的功能升级文档请查看：[FEATURES_UPGRADE_v2.0.md](./FEATURES_UPGRADE_v2.0.md)

### 技术栈

**前端：**
- Vue.js 3 / React 18
- Vite / Webpack 5
- WebSocket 实时通信
- Three.js / Babylon.js (3D 渲染)

**后端：**
- Node.js / Python FastAPI
- PostgreSQL 数据库
- Redis 缓存
- RabbitMQ / Kafka 消息队列

**DevOps：**
- Docker 容器化
- Kubernetes 编排
- GitHub Actions CI/CD
- Prometheus + Grafana 监控
